# Examples

## Python requirements

Python 3.10 or above.

Install it on a Mac with:

```shell
brew install python@3.10
```

## PDM Package & Project Manager

```shell
pdm init
pdm add jupyter
pdm install
```

## Main.py

Run it with:
```shell
python -m main --example type-hints
```
